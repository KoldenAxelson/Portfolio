{{- /* cover — build-time image optimizer for cover/badge thumbnails. Takes a cover path (as written in front matter, e.g. "/covers/foo.webp") plus a target CSS width, finds the matching SOURCE under assets/, and returns an optimized WebP resource resized for that slot — with a 2× variant for retina. All processing happens at build time (Hugo's Go image pipeline); the browser only ever receives finished, static files. Nothing client-side. ROBUSTNESS: Hugo's image pipeline can only resize RASTER formats it can decode (PNG / JPEG / WebP / GIF / TIFF / BMP). It CANNOT process SVG (vector) or AVIF — calling .Resize/.Fill/.Fit on those errors out and breaks the whole build. This template ships SVG example covers, so we guard: we only run the pipeline when the resolved resource exists AND is a processable raster. Otherwise we return the ORIGINAL path (base-path aware via func/url) with an empty srcset, so the caller renders the source image unchanged and SVG/AVIF/missing covers degrade gracefully instead of failing the build. Sources must live under assets/ (Hugo cannot read static/). A leading "/covers/" or "/badges/" path maps to the same folder under assets/; ".avif" is normalized to ".webp". Params (dict): path — cover path string from front matter (required) width — target rendered width in px (required; 2× emitted for srcset) height — target rendered height in px (optional). When set, the image is cropped to exactly width×height (Hugo Fill) — use for the square thumbnails. When omitted, the image is scaled by width preserving its aspect ratio (Hugo Resize) — use for the landscape strip. fit — when true with width+height set, scales to fit INSIDE the box without cropping (Hugo Fit) — use for badges, which shouldn't be cropped. Ignored unless height is also set. q — WebP quality 0–100 (optional, default 80) Returns a dict: { src, srcset, width, height }. For the raster path all four are populated; for the SVG/AVIF/missing fallback, srcset/width/height are "". Callers MUST treat an empty srcset as "omit the srcset attribute". {{ $c := partial "func/cover.html" (dict "path" .Params.cover "width" 120) }} {{ with $c.src }}{{ end }} */ -}} {{- $path := .path -}} {{- $width := int .width -}} {{- $height := .height -}} {{- $fit := .fit | default false -}} {{- $q := .q | default 80 -}} {{- $out := dict -}} {{- with $path -}} {{- /* Map front-matter path → assets/ source. Strip leading slash, swap .avif → .webp (AVIF sources are pre-converted to webp in assets/). */ -}} {{- $rel := strings.TrimPrefix "/" . -}} {{- $rel = replace $rel ".avif" ".webp" -}} {{- $res := resources.Get $rel -}} {{- $sub := "" -}} {{- with $res -}}{{- $sub = .MediaType.SubType -}}{{- end -}} {{- $processable := and $res (not (eq $sub "svg")) -}} {{- if $processable -}} {{- $1x := "" -}} {{- $2x := "" -}} {{- if $height -}} {{- $h := int $height -}} {{- if $fit -}} {{- $1x = $res.Fit (printf "%dx%d webp q%d" $width $h $q) -}} {{- $2x = $res.Fit (printf "%dx%d webp q%d" (mul $width 2) (mul $h 2) $q) -}} {{- else -}} {{- $1x = $res.Fill (printf "%dx%d webp q%d" $width $h $q) -}} {{- $2x = $res.Fill (printf "%dx%d webp q%d" (mul $width 2) (mul $h 2) $q) -}} {{- end -}} {{- else -}} {{- $1x = $res.Resize (printf "%dx webp q%d" $width $q) -}} {{- $2x = $res.Resize (printf "%dx webp q%d" (mul $width 2) $q) -}} {{- end -}} {{- $out = dict "src" $1x.RelPermalink "srcset" (printf "%s 1x, %s 2x" $1x.RelPermalink $2x.RelPermalink) "width" $1x.Width "height" $1x.Height -}} {{- else -}} {{- /* SVG / AVIF / missing source: pass the original path through unchanged (base-path aware) so the caller still renders it. No srcset/dims. */ -}} {{- $out = dict "src" (partial "func/url.html" $path) "srcset" "" "width" "" "height" "" -}} {{- end -}} {{- end -}} {{- return $out -}}

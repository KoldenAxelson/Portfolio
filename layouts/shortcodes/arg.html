{{- /* arg — render an argument in standard form: numbered premises, a dividing line, then the conclusion. Matches the book's "standard form" definition (premises listed first, separated from the conclusion by a horizontal line). Usage — one premise per line, a line of --- , then the conclusion: {{< arg title="Standard Form Example" >}} Whoever wrote *Macbeth* is the best playwright who ever lived. Shakespeare wrote *Macbeth*. --- Shakespeare is the best playwright who ever lived. {{< /arg >}} Optional attributes: title — caption shown above the block kind — a small label chip, e.g. "Deductive" / "Inductive" / "Valid" note — a muted footnote under the block (e.g. "Valid, but unsound.") Premise/conclusion text is rendered as inline Markdown, so *italics*, **bold**, and links work. Presentational only for now; the data-arg hook is here so it can gain interactivity later without changing any chapter file. Layout: each row is a flexbox so the P1/P2/∴ label stays inline with its statement (label is shrink-0; wrapped lines hang-indent under the text). */ -}} {{- $title := .Get "title" -}} {{- $kind := .Get "kind" -}} {{- $note := .Get "note" -}} {{- /* Split premises from conclusion on a line that is only dashes. */ -}} {{- $canon := replaceRE `(?m)^[ \t]*-{3,}[ \t]*$` "@@ARGSEP@@" (trim .Inner "\n") -}} {{- $halves := split $canon "@@ARGSEP@@" -}} {{- $prem := slice -}} {{- range (split (index $halves 0) "\n") -}} {{- with (trim . " \t") }}{{ $prem = $prem | append . }}{{ end -}} {{- end -}} {{- $concl := "" -}} {{- if gt (len $halves) 1 -}} {{- range (split (index $halves 1) "\n") -}} {{- with (trim . " \t") }}{{ if not $concl }}{{ $concl = . }}{{ end }}{{ end -}} {{- end -}} {{- end -}}
